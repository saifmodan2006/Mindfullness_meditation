# Figma Prototype Submission (Task-2)

Create a Figma design file with at least 8 to 10 screens matching the website topic.

## Suggested Frame List
1. Home Screen
2. Guided Meditation Screen
3. Mindfulness Exercises Screen
4. Relaxation Techniques Screen
5. Progress Tracker Screen
6. Resource Hub Screen
7. About Screen
8. Contact Screen
9. Login Screen
10. Sign Up Screen

## What to Upload in This Folder
- Exported PNG/JPG screens from Figma
- A PDF export of all frames (optional but recommended)
- Figma share link text file

## Naming Convention
- `01-home.png`
- `02-guided-session.png`
- `03-exercises.png`
...

This keeps your Task-2 folder clean and easy to evaluate.
